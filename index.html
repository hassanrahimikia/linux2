درسنامه جامع آزمایشگاه سیستم عامل

مجموعه دستورات و نکات کاربردی ترم جاری

۱. ناوبری و کار با دایرکتوری‌ها (Navigation)

cd /

رفتن به دایرکتوری ریشه (Root Directory). این اصلی‌ترین پوشه در لینوکس است.

cd ~ یا cd /home

رفتن به دایرکتوری Home. علامت ~ میانبری برای پوشه خانگی کاربر فعلی است.

cd ..

یک مرحله به عقب برمی‌گردد (به دایرکتوری والد).

cd ../..

دو مرحله به عقب برمی‌گردد.

pwd

Print Working Directory: مسیر کامل جایی که الان در آن هستید را نمایش می‌دهد.

ls -l

لیست کردن فایل‌ها و پوشه‌ها با جزئیات کامل (مجوزها، سایز، تاریخ).

ls -la یا ls -lah

نمایش تمام فایل‌ها حتی فایل‌های مخفی (Hidden Files که با نقطه شروع می‌شوند). سوییچ -h حجم‌ها را خوانا (Human readable) می‌کند.

mkdir [name]

Make Directory: ساخت یک پوشه جدید.

mkdir -p test/dir2/dir3

ساخت پوشه‌های تو در تو به صورت یکجا. اگر پوشه‌های والد وجود نداشته باشند، ساخته می‌شوند.

tree

نمایش ساختار فایل‌ها و پوشه‌ها به صورت درختی. (نیاز به نصب دارد).

۲. مدیریت و ویرایش فایل‌ها (File Manipulation)

touch salam.txt

ساخت یک فایل متنی خالی جدید. اگر فایل وجود داشته باشد، تاریخ ویرایش آن را بروز می‌کند.

rm -rf dir2

حذف پوشه و محتویات آن. -r (Recursive) برای حذف پوشه و -f (Force) برای حذف بدون سوال پرسیدن است. با احتیاط استفاده شود!

mv salam.txt dir2/name.txt

Move: انتقال فایل salam.txt به پوشه dir2 و تغییر نام همزمان آن به name.txt.

cat salam.txt

نمایش محتویات فایل متنی در ترمینال.

echo "salam"

چاپ کردن عبارت salam در خروجی استاندارد (ترمینال).

echo "salam" > hello.txt

Redirection (Overwrite): عبارت salam را داخل فایل hello.txt می‌ریزد. اگر فایل باشد، محتویات قبلی پاک می‌شود.

sudo sh -c "ls -lha >> /home/salam.txt"

Redirection (Append): خروجی دستور را به انتهای فایل اضافه می‌کند (بدون پاک کردن محتوای قبلی). استفاده از sh -c برای رفع مشکل Permission denied هنگام ریدایرکت با sudo است.

۳. پردازش متن و اطلاعات (Text Processing)

wc salam.txt

Word Count: نمایش تعداد خطوط، کلمات و کاراکترها.

  • wc -l: فقط تعداد خطوط
  • wc -w: فقط تعداد کلمات
  • wc -c: فقط تعداد کاراکترها
sort salam.txt

مرتب‌سازی خطوط فایل به صورت الفبایی.

uniq salam.txt

حذف خطوط تکراری پشت سر هم. (معمولا قبلش باید فایل sort شده باشد).

cat salam.txt | sort

Piping: خروجی دستور اول (cat) را به عنوان ورودی به دستور دوم (sort) می‌دهد.

nano salam.txt

باز کردن ویرایشگر متنی ساده Nano.

vim salam.txt

باز کردن ویرایشگر پیشرفته Vim.

  • خروج بدون ذخیره: :q!
  • ذخیره و خروج: :wq
  • برای نوشتن دکمه i و برای دستور دادن دکمه Esc را بزنید.

۴. مدیریت کاربران و دسترسی‌ها (Permissions & Users)

مفهوم کدها (rwx):

  • r (Read) = 4: اجازه خواندن
  • w (Write) = 2: اجازه نوشتن
  • x (Execute) = 1: اجازه اجرا کردن

مجموع این اعداد سطح دسترسی را می‌سازد (مثلا ۷ = ۴+۲+۱ یعنی دسترسی کامل).

sudo -i

ورود به محیط کاربری Root (Super User).

sudo chmod 744 amir

تغییر سطح دسترسی فولدر/فایل amir. (مالک: کامل(7) | گروه: خواندن(4) | دیگران: خواندن(4)).

sudo chmod -R 744 hoda

سوییچ -R تغییرات را روی خود پوشه و تمام فایل‌های داخل آن اعمال می‌کند.

sudo adduser roji

ساخت یک کاربر جدید به نام roji.

id rojina

نمایش ID کاربر و گروه‌های متعلق به کاربر rojina.

cat /etc/passwd

مشاهده لیست تمام کاربران سیستم و اطلاعات آنها.

۵. مدیریت سیستم و شبکه (System & Network)

sudo apt update

بروزرسانی لیست مخازن و پکیج‌ها.

sudo apt install tree

نصب برنامه جدید (در اینجا tree).

ip a یا ifconfig

نمایش وضعیت کارت‌های شبکه و آدرس IP.

df -h

نمایش فضای دیسک و پارتیشن‌ها به صورت خوانا (Disk Free).

wsl --install

دستور نصب لینوکس روی ویندوز (Windows Subsystem for Linux).
