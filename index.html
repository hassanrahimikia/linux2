مقدمه

این درسنامه شامل تمامی دستورات پایه و پیشرفته لینوکس است که در آزمایشگاه سیستم عامل نیاز دارید. هر دستور با توضیحات کامل، مثال‌های کاربردی و نکات مهم ارائه شده است.

نکته: برای اجرای دستوراتی که نیاز به دسترسی مدیریتی دارند، از sudo استفاده کنید.

مدیریت مسیرها و ناوبری

cd / - رفتن به پوشه روت

cd /

توضیح: این دستور شما را مستقیماً به ریشه سیستم فایل منتقل می‌کند. پوشه روت بالاترین سطح در ساختار درختی لینوکس است.

مثال کاربردی: 
$ pwd
/home/user/documents
$ cd /
$ pwd
/

cd .. - بازگشت به پوشه قبلی

cd ..

توضیح: یک سطح به عقب برمی‌گردد. در ساختار درختی فایل، به پوشه والد منتقل می‌شوید.

cd ../.. - بازگشت دو سطح به عقب

cd ../..

توضیح: دو سطح در ساختار درختی به عقب برمی‌گردد.

cd ~ - رفتن به پوشه خانگی

cd ~

توضیح: شما را مستقیماً به Home Directory کاربر جاری منتقل می‌کند.

cd /home - رفتن به پوشه home

cd /home

توضیح: به پوشه home در ریشه سیستم منتقل می‌شوید.

pwd - نمایش مسیر فعلی

pwd

توضیح: Print Working Directory - مسیر کامل پوشه فعلی را نمایش می‌دهد.

مدیریت پوشه‌ها

mkdir - ساخت پوشه جدید

mkdir folder_name

توضیح: Make Directory - یک پوشه جدید ایجاد می‌کند.

mkdir -p - ساخت پوشه‌های تو در تو

mkdir -p test/dir2/dir3

توضیح: پرچم -p به شما اجازه می‌دهد چندین پوشه تو در تو را همزمان بسازید.

ls - نمایش محتویات

ls

توضیح: لیست فایل‌ها و پوشه‌ها را نمایش می‌دهد.

ls -l - نمایش جزئیات

ls -l

توضیح: محتویات را به صورت لیستی با جزئیات نمایش می‌دهد شامل مجوزها، مالک، اندازه و تاریخ.

ls -la - نمایش فایل‌های مخفی

ls -la

توضیح: تمام فایل‌ها از جمله فایل‌های مخفی را نمایش می‌دهد.

ls -lah - نمایش با واحد خوانا

ls -lah

توضیح: اندازه فایل‌ها را به صورت خوانا نمایش می‌دهد.

rm -rf - حذف پوشه

rm -rf dir2

توضیح: پوشه و تمام محتویات آن را بدون تأیید حذف می‌کند.

هشدار: این دستور بسیار خطرناک است و غیرقابل بازگشت می‌باشد!

tree - نمایش ساختار درختی

tree

توضیح: ساختار پوشه‌ها را به صورت درختی نمایش می‌دهد.

نصب:
sudo apt update
sudo apt install tree

مدیریت فایل‌ها

touch - ساخت فایل

touch salam.txt

توضیح: یک فایل خالی ایجاد می‌کند.

echo - نمایش متن

echo "salam"

توضیح: متن را در ترمینال چاپ می‌کند.

echo > - نوشتن در فایل

echo "salam" > hello.txt

توضیح: متن را داخل فایل می‌نویسد و محتوای قبلی را پاک می‌کند.

cat - نمایش محتوای فایل

cat salam.txt

توضیح: محتوای کامل فایل را نمایش می‌دهد.

nano - ویرایشگر nano

nano salam.txt

توضیح: فایل را با ویرایشگر nano باز می‌کند.

vim - ویرایشگر vim

vim salam.txt

توضیح: فایل را با ویرایشگر vim باز می‌کند.

دستورات vim:
- ESC: خروج از حالت ویرایش
- :q! خروج بدون ذخیره
- :wq ذخیره و خروج

mv - انتقال و تغییر نام

mv salam.txt dir2/name.txt

توضیح: فایل را منتقل کرده و نام آن را تغییر می‌دهد.

wc - شمارش

wc salam.txt

توضیح: تعداد خطوط، کلمات و کاراکترها را نمایش می‌دهد.

wc -l - شمارش خطوط

wc -l salam.txt

توضیح: فقط تعداد خطوط را نمایش می‌دهد.

wc -w - شمارش کلمات

wc -w salam.txt

توضیح: فقط تعداد کلمات را نمایش می‌دهد.

wc -c - شمارش کاراکترها

wc -c salam.txt

توضیح: فقط تعداد کاراکترها را نمایش می‌دهد.

مجوزها و دسترسی‌ها

درک سیستم مجوزها

در لینوکس هر فایل دارای مجوزهای rwx است:

  • r (Read) = 4 - خواندن
  • w (Write) = 2 - نوشتن
  • x (Execute) = 1 - اجرا

مجموع اعداد مجوز را مشخص می‌کند:

  • 7 = rwx (همه مجوزها)
  • 6 = rw- (خواندن و نوشتن)
  • 5 = r-x (خواندن و اجرا)
  • 4 = r-- (فقط خواندن)

chmod - تغییر مجوزها

sudo chmod 744 amir

توضیح: مجوزهای دسترسی را تغییر می‌دهد. 744 یعنی: مالک (rwx)، گروه (r)، دیگران (r)

chmod -R - تغییر بازگشتی

sudo chmod -R 744 hoda

توضیح: مجوزها را برای پوشه و تمام محتویات آن تغییر می‌دهد.

sudo -i - ورود به root

sudo -i

توضیح: به حساب کاربری root وارد می‌شوید.

هشدار: در حالت root تمام دسترسی‌ها را دارید. با احتیاط عمل کنید!

مدیریت کاربران

sudo adduser - ساخت کاربر جدید

sudo adduser roji

توضیح: یک کاربر جدید با نام مشخص شده ایجاد می‌کند.

passwd - تغییر رمز عبور

passwd

توضیح: رمز عبور کاربر فعلی را تغییر می‌دهد.

id - اطلاعات کاربر

id rojina

توضیح: شناسه کاربر و گروه را نمایش می‌دهد.

cd /etc - پوشه تنظیمات

cd /etc

توضیح: به پوشه etc که شامل فایل‌های پیکربندی سیستم است می‌رود.

cat /etc/passwd - لیست کاربران

cat /etc/passwd

توضیح: لیست تمام کاربران سیستم را نمایش می‌دهد.

exit - خروج

exit

توضیح: از حساب کاربری فعلی خارج می‌شوید (log out).

پردازش متن

uniq - حذف تکراری‌ها

uniq salam.txt

توضیح: خطوط تکراری پشت سر هم را حذف کرده و یکی نگه می‌دارد.

نکته: این دستور فقط خطوط پشت سر هم را بررسی می‌کند.

sort - مرتب‌سازی

sort salam.txt

توضیح: خطوط فایل را به ترتیب الفبایی مرتب می‌کند.

cat | sort - ترکیب دستورات

cat salam.txt | sort

توضیح: ابتدا محتوا را مرتب کرده سپس نمایش می‌دهد. عملگر | خروجی یک دستور را به ورودی دستور بعدی می‌فرستد.

اطلاعات سیستم

ip a - اطلاعات شبکه

ip a

توضیح: لیست کارت‌های شبکه و آدرس‌های IP را نمایش می‌دهد.

ifconfig - پیکربندی شبکه

ifconfig

توضیح: اطلاعات کارت‌های شبکه را به ترتیب نمایش می‌دهد.

df -h - فضای دیسک

df -h

توضیح: میزان استفاده از فضای دیسک را به صورت خوانا نمایش می‌دهد.

sudo apt update - به‌روزرسانی لیست پکیج‌ها

sudo apt update

توضیح: لیست پکیج‌های موجود را به‌روزرسانی می‌کند.

دستورات پیشرفته

sudo sh -c - اجرای دستورات پیچیده با sudo

sudo sh -c "ls -lah >> /home/salam.txt"

توضیح: برای redirect کردن خروجی به فایل‌های سیستمی که نیاز به دسترسی root دارند.

WSL --install - نصب WSL

WSL --install

توضیح: در ویندوز، زیرسیستم لینوکس را نصب می‌کند.

نکات مهم و خلاصه

  • همیشه قبل از اجرای دستورات خطرناک مانند rm -rf دقت کنید
  • از sudo فقط زمانی استفاده کنید که واقعاً نیاز دارید
  • برای یادگیری بهتر، دستورات را خودتان تمرین کنید
  • از man command برای دیدن راهنمای کامل هر دستور استفاده کنید
  • قبل از تغییر مجوزها، از تأثیر آن مطمئن شوید
